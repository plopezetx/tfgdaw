# 00 - Contexto del Proyecto

## Nombre provisional

IDE Web Colaborativo con Asistente IA

## Descripción general

El proyecto consiste en desarrollar una plataforma web con un IDE integrado que permita a los usuarios crear, editar, guardar, ejecutar, publicar y reutilizar proyectos de código desde el navegador.

El sistema estará orientado inicialmente a proyectos web basados en HTML, CSS, JavaScript, TypeScript y frameworks modernos de frontend. El objetivo principal es reducir la fricción de configuración local y permitir que una persona pueda empezar a programar, ejecutar y compartir un proyecto desde una única plataforma web.

## Motivación

Actualmente, muchas personas que están aprendiendo desarrollo web o creando prototipos pequeños se enfrentan a varios problemas:

- Necesidad de instalar entornos locales.
- Dificultad para compartir proyectos ejecutables.
- Separación entre edición, ejecución, almacenamiento y publicación.
- Falta de herramientas integradas de ayuda al desarrollo en entornos educativos.

Este TFG busca centralizar esas capacidades en una plataforma web sencilla, usable y técnicamente defendible.

## Alcance general

El alcance recomendado del proyecto se centra en:

- Editor de código integrado.
- Gestión de proyectos por usuario.
- Ejecución local en navegador.
- Vista previa del resultado.
- Publicación de proyectos.
- Galería pública.
- Fork o remix de proyectos.
- Asistente IA modular como ampliación de valor.

## Fuera de alcance inicial

No forman parte del MVP inicial:

- Ejecución generalista de cualquier lenguaje.
- Terminal Linux persistente en servidor.
- Clon completo de VS Code.
- Marketplace de extensiones.
- Ejecución de código no confiable en infraestructura propia.
- Colaboración en tiempo real completa tipo Google Docs.

## Enfoque del TFG

El proyecto se plantea como una plataforma browser-native, desktop-first y orientada a proyectos web. La ejecución principal se realizará en el navegador para reducir riesgos de seguridad y costes de infraestructura.
